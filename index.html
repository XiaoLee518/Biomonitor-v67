關於我們
本系統由學生團隊自主研發，採用 ESP32 微控制器搭載 MAX30102 光學心率感測器、GSR 膚電模組與應變規呼吸感測器，透過 BLE 無線傳輸與 Web Bluetooth API 實現即時生理數據可視化。

本系統整合動態閾值心率偵測、斜率式呼吸偵測與膚電反應分析，提供即時壓力指數量化監測。

壓力指數公式
Score_HR = (Current_BPM / Base_BPM) × 1.20
Score_RPM = (Current_RPM / Base_RPM) × 1.25
Score_GSR = (Current_GSR / Base_GSR) × 0.80
Stress = Score_HR×0.4 + Score_RPM×0.4 + Score_GSR×0.2
基準值 = 1.14（靜息狀態） · 越高 = 越緊張

緊張判定閾值預覽
S_HR = (BPM − restHR) / (restHR × 40%) × 100  → 100 當 BPM 超出靜息 40%（靜息70 → 滿分98 BPM）
S_GSR = (baseGSR − gsrRaw) / (baseGSR × 0.20) × 100  → 100 當感測器值相對基準下降 20%（電導上升）
S_Resp = (RPM − calmResp) / (calmResp × (1.20 − 1)) × 100  → 100 當 RPM 達到基準 × 1.20
Stress = [(BPM−Base)/Base×0.20]×0.4 + [(RPM−Base)/Base×0.25]×0.4 + [(Base−GSR)/Base×0.20]×0.2 ×100
LTI = Peak×0.4 + Avg×0.4 + 事件密度×0.2  （報告頁顯示）
